## 📁 Descripción del Repositorio
Este repositorio contiene el proyecto final de la materia **Programación 2**.  
El objetivo es aplicar los conceptos de **Programación Orientada a Objetos en Java**, desarrollando un sistema de gestión que permite administrar **categorías, productos, usuarios y pedidos** mediante un menú principal en consola.  

Se implementan operaciones CRUD completas y relaciones entre entidades, garantizando un código ordenado y mantenible.

---

## 🧱 Estructura del Repositorio
- `entities/` → Clases de entidades (Usuario, Producto, Pedido, etc.)  
- `services/` → Servicios con lógica de negocio y validaciones  
- `enums/` → Enumeraciones (Rol, Estado, FormaPago)  
- `Main.java` → Clase principal con menú interactivo  

---

## 🚀 Tecnologías Utilizadas
- **Java JDK 21**  
- **NetBeans 29**  
- Sistema operativo: Windows / Linux / macOS  

---

## 📌 Objetivos Académicos
- Aplicar conceptos de **POO** (clases, objetos, herencia, polimorfismo).  
- Implementar operaciones CRUD en consola.  
- Gestionar relaciones entre entidades (usuarios ↔ pedidos ↔ productos).  
- Fomentar buenas prácticas de programación y documentación.
